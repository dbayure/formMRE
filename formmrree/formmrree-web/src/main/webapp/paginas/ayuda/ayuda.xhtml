Ayuda del Sistema

Acceso al Sistema, el usuario debe ingresar usuario y clave. El sistema se encarga de validar las credenciales con la base de datos.

El usuario administrador debe ser ingresado directamente en la base, su constraseña debe ser creada en sha 256 con 64 bits

Página principal del sistema, aqui se puede ver el menú de opciones y el listado de formularios que hayan sido ingresados

En la opción "Formulario" del menú se puede acceder al ingreso una nuevo formulario con la cantidad de personas que se requiera, además se puede visualizar los formularios que hayan sido ingresados

En la opción "Usuario" del menú se pueden agregar roles y usuarios del sistema, es necesario configurar el servidor para que los nuevos roles queden activos. Los usuarios creados deben tener un rol asignado

En la opción "Administración Sistema" se pueden igresar los catálogos del sistema, estos son necesarios para completar la iformación de ingreso de cada persona, deben estar creados previamente al ingreso de cada formulario

En la esquina superior derecha de la pantalla se puede ver cual es el usuario que ingresó al sistema, también es posible terminar la sesión con el botón logout

Una vez ingresados los formularios de las familias que rotornan al país, es posible visualizar en detalle cada una de ellas, para realizar esa tarea basta con presionar la lupa que se encuentra a la derecha de cada formulario y debajo se despliega la lista de las personas que pertenecen al mismo Si se quiere ver el detalle de una persona o modificar alguno de sus datos, se puede presionar el botón con forma de llave que se encuentra a la derecha de la misma, si se quiere eliminar la persona se oprime el botón conforma de papelera.

Se despliega la información de la persona, los campos que pueden ser modificados quedan habilitados y al final de la página se muestra un botón par aconfirmar la acción
